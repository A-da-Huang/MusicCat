Musical Cat is an interactive, keyboard-controlled soundboard built with p5.js that combines sound effects with simple image-based animation. Each key press triggers a sound while visually animating the cat, creating a playful and responsive experience that blends creativity with code.

This project was created as a fun way to explore event-driven programming, audio playback, and visual feedback in the browser using JavaScript. Also as my final project for my Creative Coding course.

p5.js makes creative coding accessible and beginner-friendly. It abstracts away complex browser APIs and allows developers to focus on interaction, visuals, and experimentation.

For this project, p5.js was ideal because it:

Simplifies keyboard input handling

Makes audio playback intuitive

Encourages rapid prototyping and creativity

# Overview
Musical Cat turns your keyboard into an instrument by pressing a key, and the cat reacts.

At its core, this project listens for keyboard input and maps specific keys to:

Sound effects (loaded from local audio files)

Animated cat images that change based on interaction.

The goal wasnâ€™t just to play sounds, but to make the interaction feel alive by pairing audio with visual motion. By cutting and swapping image frames, the cat appears to move its arms when sounds are triggered.

This project demonstrates how simple animations and audio cues can significantly improve user experience, even in small projects.
